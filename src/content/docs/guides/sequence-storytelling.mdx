单张照片解决的是“这一张是否成立”。

组图解决的是另一个问题：

**这些照片放在一起，能不能形成一种观看经验？**

第 18 章会带你做 lofi 个人系列，这一页补更通用的组图叙事方法。以后你拍街头、旅行、房间、朋友、夜路、相机项目，都能用。

## 这一页学完以后，你应该能做到

- 区分单张强和组图强
- 给一组照片安排开场、中段和结尾
- 用重复和变化建立系列感
- 知道为什么不是每张都要当封面
- 提交照片时给我一个更好点评的顺序

## 先记住一句话

**组图不是把好照片堆起来，而是让照片之间发生关系。**

关系可以来自：

- 同一个地点
- 同一种颜色
- 同一种距离
- 同一个人物状态
- 同一种光线
- 同一种情绪
- 一个从开始到结束的变化

## 组图最基础的 5 个位置

你可以把一组 8 到 12 张照片想成一段短片：

| 位置 | 作用 | 常见照片类型 |
| --- | --- | --- |
| 开场 | 设定气氛 | 空场、路口、窗、远景、安静的第一张 |
| 进入 | 让观众知道在哪里 | 环境、细节、人物出现前的线索 |
| 展开 | 形成主题 | 人物、动作、重复的场景 |
| 峰值 | 最强的一两张 | 情绪最高、结构最强、瞬间最明确 |
| 结尾 | 留下余味 | 空场、背影、暗灯、远去、安静细节 |

不是每组都要讲故事，但每组都需要顺序。

## 重复和变化

系列感来自重复，但如果只有重复，会无聊。

你需要同时设计：

- 重复：让观众知道这是同一组
- 变化：让观众愿意继续看下去

比如：

| 重复什么 | 可以变化什么 |
| --- | --- |
| 都是夜晚 | 从空街到人物，再回到空街 |
| 都是低饱和 | 亮暗节奏有变化 |
| 都是公交站 | 远景、近景、人物、细节交替 |
| 都是失焦 | 有些全糊，有些保留清楚锚点 |
| 都是朋友 | 表情、距离、环境变化 |

## 选片：别只选最猛的

一组照片里需要不同角色：

- 封面图：最能代表这组
- 过渡图：不一定最强，但让节奏顺
- 信息图：交代地点、时间、人物关系
- 情绪图：不一定信息多，但气氛准
- 结尾图：让人停一下

如果每张都很用力，整组会像一首歌全程副歌，没有呼吸。

## 最简单的排序方法

你可以先用这个顺序：

1. 选 20 张候选
2. 删到 12 张
3. 找出最像封面的 1 张
4. 找出最适合结尾的 1 张
5. 中间按“远景、近景、人物、细节、强图、安静图”穿插
6. 看一遍，删掉重复但没有新信息的照片

最后问：

- 第一张有没有把人带进去？
- 中段有没有变化？
- 最强的照片是不是出现得太早？
- 结尾有没有余味？
- 有没有哪张只是在重复上一张？

## 常见错误 vs 推荐做法

    <ul>
      <li>主题跳来跳去</li>

    <ul>
      <li>开场建立气氛</li>
      <li>中段制造变化</li>
      <li>结尾留下余味</li>
    </ul>
  </DoDontCard>
  <DoDontCard mood="bad" title="舍不得删重复照片" intro="相似照片太多，会让组图变松。">

      <li>同一情绪没有推进</li>
      <li>观众很快疲劳</li>
    </ul>
  </DoDontCard>
  <DoDontCard mood="good" title="用重复建立风格，用变化建立节奏" intro="这是系列成立的核心。">

      <li>变化距离和场景</li>

    </ul>
  </DoDontCard>

## 组图作业怎么拍

<PracticeCard title="组图叙事作业" intro="这次重点不是单张，而是顺序和关系。">
  <ul>
    <li>选一个主题，拍 20 张，最后只留 10 到 12 张。</li>
    <li>给这组照片起一个名字，并写一句主题句。</li>
    <li>按开场、进入、展开、峰值、结尾排序。</li>
    <li>提交时告诉我：你删掉了哪类重复照片，为什么。</li>
  </ul>
</PracticeCard>

<BridgeNote title="换成相机时怎么迁移" intro="组图叙事完全不依赖设备。">
  <ul>
    <li>相机会提高单张质量，但不会自动让系列成立。</li>
    <li>以后做旅行、街拍、人像、家庭记录或长期项目，都需要组图思维。</li>
    <li>你现在用手机练顺序和删片，是非常重要的长期能力。</li>
  </ul>
</BridgeNote>

## 这一页最重要的一句话

**单张靠瞬间，组图靠关系。**
