后期不是把废片救成神图。

更现实、更有用的理解是：

**后期是在完成一张已经有基础的照片。**

## 基础后期顺序

建议先按这个顺序练：

1. 裁切
2. 曝光
3. 对比
4. 白平衡
5. 色彩
6. 细节
7. 颗粒或风格化

顺序很重要。太早拉饱和、清晰度、锐化，很容易让照片看起来“变化很大”，但不是更好。

## 裁切：经常比滤镜更有效

裁切的意义不是“后期修补”，而是重新明确画面重点。

一张照片常常只要：

- 去掉边缘干扰
- 调整一点上下比例
- 收掉没必要的空白
- 让主体的位置更明确

就会比乱加滤镜有效得多。

## 曝光和对比：把气氛拉回正确方向

这一层你要问自己的不是：

- “亮不亮”

而是：

- “它现在像不像我想要的氛围”

比如：

- 夜景如果被提得太亮，氛围就没了
- 人像如果太暗，脸就会失去舒服感
- 静物如果对比太高，质感会变脏

## 白平衡和色彩：先统一，再表达

很多照片后期翻车，都是因为颜色过多、过艳、没有统一方向。

更稳的思路通常是：

- 一组照片先定一个冷暖倾向
- 再看主色是不是统一
- 最后才微调局部颜色

如果你想系统学颜色，看：[色彩基础](../color-foundations/)。

## 细节：最后再碰，而且要克制

锐化、清晰、结构这类选项最容易让照片看起来“有效果”，但也最容易把照片毁掉。

一旦过度，常见问题就是：

- 边缘发硬
- 皮肤发脏
- 灯光发刺
- 细节像被涂了一层塑料

所以这类选项一定是最后再看，而且宁可少一点。

## 后期作业怎么做

<PracticeCard title="基础后期作业" intro="同一张图做不同版本，最容易看出后期的作用。">
  <ul>
    <li>选 3 张旧照片，只做裁切，不调颜色，比较前后差异。</li>
    <li>同一张夜景做正常、压暗、偏亮 3 个版本，比较氛围。</li>
    <li>做 1 组 6 张照片，统一白平衡和饱和度，要求像同一组。</li>
  </ul>
</PracticeCard>

<BridgeNote title="换成相机时怎么迁移" intro="软件会变，顺序不会变。">
  <ul>
    <li>以后用 Lightroom、Capture One 或其他软件，也仍然先裁切、亮暗、色彩、细节。</li>
    <li>相机 RAW 文件空间更大，但也更考验克制。</li>
    <li>现在先用手机建立顺序，比先学复杂软件更重要。</li>
  </ul>
</BridgeNote>

## 这一页最重要的一句话

**后期不是加效果，而是把照片收拢到你的意图上。**
