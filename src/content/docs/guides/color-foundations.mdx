构图、光线和色彩，是照片最重要的三条视觉语言。

前面章节已经讲了构图和光线，这一页专门补色彩。目标不是让你背色环，而是让你在拍摄和后期时能判断：

**这张照片的颜色是在帮忙，还是在抢戏？**

## 这一页学完以后，你应该能做到

- 分清主色、辅色和干扰色
- 知道冷暖会改变照片情绪
- 用低饱和避免廉价滤镜感
- 看懂互补色为什么容易有张力
- 拍一组颜色统一的小练习

## 先记住一句话

**好色彩不是颜色多，而是颜色有主次。**

一张照片颜色乱，通常不是因为你不会调色，而是因为拍的时候就没有控制：

- 背景颜色太抢
- 主体和环境互相打架
- 光源颜色太混
- 后期每张都往不同方向拉

## 主色、辅色、干扰色

你可以把一张照片里的颜色分成三类：

| 类型 | 作用 | 判断问题 |
| --- | --- | --- |
| 主色 | 决定照片整体气质 | 这张图第一感觉是什么颜色？ |
| 辅色 | 支撑主色，让画面不单调 | 它是在帮主色，还是在抢主角？ |
| 干扰色 | 抢注意力但没有意义 | 能不能换角度、裁掉或压低饱和？ |

拍之前可以先问：

- 这张照片的主色是什么？
- 有没有一个颜色太跳，但和主题无关？
- 主体颜色和背景颜色有没有分开？
- 如果把某个颜色拿掉，照片会不会更干净？

## 冷暖不是准确问题，而是情绪问题

冷暖会直接改变观看感受：

- 暖色：亲近、旧、室内、傍晚、回忆
- 冷色：疏离、安静、雨天、金属、城市夜晚
- 中性色：克制、普通、日常、观察感

你可以这样用：

| 方向 | 适合风格 | 手机上怎么做 |
| --- | --- | --- |
| 偏暖 | 日常、房间、midwest emo、旧记忆 | 找暖灯、傍晚光，后期略暖 |
| 偏冷 | post-punk、雨天、夜路、疏离感 | 阴天、蓝色阴影，后期略冷 |
| 冷暖混合 | 城市夜景、便利店、地铁口 | 保留现场光，但别让颜色全炸 |
| 低饱和 | lofi、Stephen Shore 式日常、粗粝街头 | 降低饱和，保留少数关键颜色 |

## 互补色：最容易制造张力

互补色可以先理解成“彼此对着干”的颜色关系。

常见组合：

- 蓝 / 橙
- 红 / 绿
- 黄 / 紫

手机拍摄里最常见的是：

- 蓝色夜晚 + 橙色路灯
- 绿色便利店光 + 红色招牌
- 冷色天空 + 暖色窗户

互补色不一定要很鲜艳。你喜欢的 lofi、post-punk、shoegaze 方向，很多时候更适合：

**低饱和互补色。**

也就是有颜色冲突，但不刺眼。

## 低饱和为什么常常更耐看

低饱和不是“没有颜色”，而是让颜色退一步，让结构、光和情绪出来。

适合低饱和的场景：

- 街头颜色很多
- 夜晚招牌复杂
- 想拍粗粝、冷感、普通日常
- 想让照片像记忆，而不是广告

不适合无脑低饱和的场景：

- 食物本来靠颜色诱人
- 明亮快乐的主题
- 颜色本身就是主体
- 需要真实肤色的人像

## 小米 15 Ultra 上怎么练色彩

拍摄时：

- 先找一个主色明显的场景
- 避免背景里出现无意义的高饱和颜色
- 夜景先压一点曝光，别让彩灯全炸
- 拍人时注意肤色，不要为了风格把脸调得太怪

后期时：

- 先统一白平衡
- 再降低整体饱和或自然饱和
- 最后只保留一个关键颜色
- 一组照片不要每张都偏不同方向

<ParamCard title="色彩练习起点" intro="不用一次学完整色彩学，先从可拍的小判断开始。">
  <ul>
    <li>单主色：整张照片只让一个颜色最明显。</li>
    <li>冷暖对比：找蓝色环境里的暖灯，或者暖色房间里的冷窗光。</li>
    <li>低饱和：把颜色压低，但保留一个视觉锚点。</li>
    <li>肤色保护：拍人时先保证脸不怪，再考虑整体风格。</li>
  </ul>
</ParamCard>

## 色彩作业怎么拍

<PracticeCard title="色彩基础作业" intro="这次不追求复杂，先把颜色主次练出来。">
  <ul>
    <li>拍 3 张单主色照片：每张只能有一个最主要颜色。</li>
    <li>拍 3 张冷暖对比照片：比如冷环境里的暖灯，或暖房间里的冷窗光。</li>
    <li>拍 1 组 6 张低饱和照片，要求颜色统一但不死灰。</li>
    <li>如果有人像，额外说明你有没有保护肤色。</li>
  </ul>
</PracticeCard>

<BridgeNote title="换成相机时怎么迁移" intro="色彩判断比设备更重要。">
  <ul>
    <li>相机可能给你更大的后期空间，但主色、冷暖和干扰色判断不变。</li>
    <li>以后拍 RAW，会有更强的调色自由，也更容易乱调。</li>
    <li>先用手机练“颜色主次”，比先学复杂调色软件更稳。</li>
  </ul>
</BridgeNote>

## 这一页最重要的一句话

**别先问怎么调色，先问这张照片到底需要哪些颜色。**
