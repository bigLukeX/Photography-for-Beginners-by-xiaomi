**你到底在取舍什么。**

## 曝光：不是亮一点暗一点

曝光最值得先学会的一句话是：

**每次曝光选择，都是在决定你更想保住什么。**

比如：

- 你想保住天空细节，就可能让地面暗一点
- 你想保住人物脸部，就可能让背景更亮一些
- 你想做夜景氛围，就不应该把一切都提得和白天一样亮

所以曝光不是“越亮越对”，而是“你有没有意识地选”。

## 动态范围：为什么不能什么都保住

动态范围可以理解成：

**一张照片同时容纳很亮和很暗细节的能力。**

现实里经常会遇到这种场景：

- 天空很亮，地面很暗
- 灯牌很亮，人脸很暗
- 窗外很亮，室内很暗
- 舞台灯很亮，观众席很暗

这时手机或相机都要做取舍。你把画面提亮，暗部出来了，但高光可能炸掉；你压暗保高光，灯牌细节保住了，但暗部可能变黑。

手机的 `HDR`、夜景模式、多帧合成，会试图帮你同时保住更多亮暗细节。但它也可能带来副作用：

- 夜景被提得太亮，氛围消失
- 暗部被抹得太干净，质感变塑料
- 高反差被拉平，画面不够有力量

所以基础判断不是“开不开 HDR”这么简单，而是：

**这张照片我更想保高光、保脸，还是保夜晚的黑？**

## 曝光三件事：进光时间、感光代价、计算补偿

传统相机常讲“曝光三角”：光圈、快门、ISO。

手机上你要稍微换一种理解：

- 光圈：多数手机镜头的物理光圈基本固定，你通常不能像相机那样随手改变
- 快门：决定进光时间，也决定运动是被定住还是被拖开
- ISO：决定暗光下用多少感光增益，越高越容易有噪点和脏细节
- 曝光补偿：在自动模式里告诉手机整体亮一点或暗一点

所以手机最常用的控制不是完整三角，而是：

**快门 + ISO + 曝光补偿 + 对焦/测光 + 后期。**

## 常用参数大概怎么读

这些不是死规定，而是你出门时可以用来判断的起点：

| 参数 | 常见范围 | 大致效果 |
| --- | --- | --- |
| `1/1000s` 到 `1/500s` | 快门很快 | 定住车、人、运动，光少时画面会暗 |
| `1/250s` 到 `1/125s` | 日常抓拍 | 普通走路、朋友动作比较稳 |
| `1/60s` 到 `1/30s` | 手持边界 | 静物还能拍，人物开始容易糊 |
| `1/15s` 到 `1/4s` | 慢门入口 | 人影、车灯、手部动作开始拖开 |
| `1s` 或更慢 | 明显慢门 | 需要稳定手机，适合车流、灯线、水面 |
| `ISO 50/100` | 干净 | 白天、强光、稳定夜景优先 |
| `ISO 400` | 轻微质感 | 暗光可用，颗粒感开始出现 |
| `ISO 800+` | 粗糙 | 夜晚、lofi、颗粒感可以接受，但细节会脏 |
| `-0.3EV/-0.7EV` | 压暗 | 保高光、保夜景氛围、减少手机过度提亮 |
| `+0.3EV/+1EV` | 提亮 | 人脸、柔光、漂浮感，但高光更危险 |

## 光圈：现在先理解，以后换相机就会用上

换成相机以后，光圈会变成一个真正可调的核心参数：

| 光圈 | 大致效果 | 常见用途 |
| --- | --- | --- |
| `f/1.4-f/2` | 进光多，景深浅，背景更容易虚 | 人像、暗光、主体突出 |
| `f/2.8-f/4` | 兼顾进光和清楚范围 | 日常、人像、街拍 |
| `f/5.6-f/8` | 清楚范围更大 | 风景、建筑、多人合影 |
| `f/11+` | 景深更深，但可能需要更慢快门或更高 ISO | 强光风景、特殊需求 |

这就是为什么你现在用手机，也要先理解“光圈会影响景深和进光”。现在你未必能调它，但以后换相机时，它会立刻变成重要控制项。

  <ul>
    <li>夜景先试 `-0.3EV/-0.7EV`，看灯牌和暗部是否更有氛围。</li>
    <li>逆光人像先保脸，再决定背景能不能亮一点。</li>
    <li>慢门先固定手机，再降低 ISO，别只靠后期救糊。</li>
    <li>高 ISO 不是错误，但要知道你是在换颗粒和粗糙。</li>
  </ul>
</ParamCard>

## 作业怎么拍

<PracticeCard title="曝光和动态范围作业" intro="这次重点不是哪张最好看，而是说清楚你保住了什么。">
  <ul>
    <li>同一夜景拍 `0EV`、`-0.3EV`、`-0.7EV` 三张，比较灯牌和暗部。</li>
    <li>同一逆光场景拍一张保脸、一张保背景，写出你接受的代价。</li>
    <li>用 `1/250s` 和 `1/15s` 分别拍有动作的主体，比较时间感。</li>
  </ul>
</PracticeCard>

<BridgeNote title="换成相机时怎么迁移" intro="相机会给你更多控制，但取舍逻辑不变。">

    <li>手机上的快门和 ISO 练习，会直接迁移到相机手动模式。</li>
    <li>相机多出的光圈，会让你更精确地控制进光和景深。</li>
  </ul>

**曝光不是让照片变亮，而是决定你愿意牺牲什么。**
