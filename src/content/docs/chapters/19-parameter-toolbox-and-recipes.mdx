这一章不是让你从头到尾背完。

它更像一个出门前、拍摄中、复盘时都能查的工具箱：你想拍清楚、拖影、颗粒、梦境感、post-punk、shoegaze，都可以先回来看对应的参数起点。

## 这一章学完以后，你应该能做到

- 看到一个场景时，知道快门、ISO、白平衡大概从哪里开始
- 知道自动模式、曝光补偿、专业模式分别适合什么时候用
- 用参数主动制造拖影、颗粒、失焦和高光溢出
- 把高级风格变成可执行的拍摄配方
- 以后换相机时，能把这些逻辑迁移过去

## 先记住一句话

**参数不是答案，是起点。**

同样的 `1/15s`，拍走路的人、车灯、雨夜玻璃、室内灯，都可能完全不一样。

所以你要学的不是“某个数字一定对”，而是：

- 这个数字大概会带来什么结果
- 如果结果不对，下一步往哪里调
- 这个代价你能不能接受

## 快门：控制时间，不只是控制清晰

快门越快，越像切下一瞬间。

快门越慢，越像把一段时间压进照片。

| 快门 | 画面效果 | 练习方向 |
| --- | --- | --- |
| `1/1000s` | 定住快速动作 | 白天车、人、运动 |
| `1/250s` | 日常抓拍稳 | 街拍、朋友、行走 |
| `1/125s` | 人像基本安全 | 站定人物、半身人像 |
| `1/60s` | 手持边界 | 静物、弱光、轻微风险 |
| `1/15s` | 人物开始拖 | 模糊记忆、夜路、经过的人 |
| `1/4s` | 明显拖影 | 车灯、人群、shoegaze 气质 |
| `1s+` | 时间流动 | 车流、水面、固定机位 |

如果你想拍清楚：往更快快门走。

如果你想拍拖影：往更慢快门走。

如果画面太暗：提高 ISO、找更亮现场光、或者接受更慢快门。

## ISO：控制暗光代价，也控制颗粒气质

ISO 不是“亮度按钮”，它更像是“我愿意用多少粗糙换暗光可见”。

| ISO | 画面倾向 | 适合 |
| --- | --- | --- |
| `50/100` | 干净、细节好 | 白天、慢门固定、干净风景 |
| `200/400` | 轻微暗光质感 | 室内、阴天、傍晚 |
| `800` | 噪点开始明显 | 夜景、街头、lofi |
| `1600+` | 粗糙、涂抹、颗粒强 | post-punk、森山大道式粗粝、故意不干净 |

你喜欢的颗粒感，不一定要完全靠后期加。暗光、高 ISO、压暗曝光、后期颗粒，可以一起形成质感。

## 曝光补偿：自动模式里最重要的控制

很多时候你不需要进专业模式，只需要曝光补偿。

| 曝光补偿 | 适合什么 | 风险 |
| --- | --- | --- |
| `-1EV` | 强灯牌、夜景、高反差 | 暗部可能死黑 |
| `-0.7EV` | 夜景保高光 | 人脸可能偏暗 |
| `-0.3EV` | 日常稍微压住亮部 | 整体略暗 |
| `0EV` | 自动判断可接受 | 手机可能把夜景提太亮 |
| `+0.3EV` | 柔光、人脸、阴天 | 高光风险增加 |
| `+1EV` | 漂浮、过曝边缘、梦境感 | 细节容易丢 |

你要养成一个习惯：夜景和高反差场景，先试 `-0.3EV` 或 `-0.7EV`。

## 白平衡：不是只追求准确

白平衡决定照片像什么温度的记忆。

| 色温 | 视觉感受 | 可以对应的风格 |
| --- | --- | --- |
| `2800K-3500K` | 暖、旧、路灯、室内 | lofi、midwest emo、夜间日常 |
| `4000K-5000K` | 灰、冷暖混合、普通 | Stephen Shore 式日常观察 |
| `5200K-6000K` | 相对准确 | 白天、真实肤色、基础练习 |
| `6500K+` | 冷、蓝、疏离 | post-punk、雨天、梦境、冷感城市 |

如果你拍的是作业和基础练习，先保持相对准确。

如果你拍的是高级篇，可以让白平衡偏离现实，但偏离要统一。

## 对焦、失焦和测光

对焦控制清楚，测光控制亮暗参考。

你可以这样理解：

- 想拍清楚：点主体对焦
- 想保脸：点脸部测光
- 想保灯牌：点灯牌附近测光，必要时压暗
- 想拍失焦：点主体旁边、前景或背景，让主体轻微偏离焦点
- 想等人进画面：先对焦锁定在某个距离，再等人走进来

失焦类照片要保留一个锚点：

- 轮廓
- 姿态
- 光源
- 影子
- 背景结构

否则它不像表达，只像失误。

## 风格配方

### post-punk 城市夜晚

- 场景：地铁口、楼道、停车场、便利店门口
- 焦段：`1x` 或轻长焦
- 快门：`1/60s-1/250s`
- ISO：`400-1600`
- 曝光：`-0.3EV/-0.7EV`
- 后期：低饱和、高反差、允许颗粒

### midwest emo 日常空场

- 场景：房间、窗边、公交站、空椅子、傍晚路口
- 焦段：`1x`
- 快门：`1/125s-1/250s`
- ISO：`50-400`
- 曝光：`0EV` 或略暗
- 后期：低饱和、低对比一点、保留普通细节

### dreampop 柔光漂浮

- 场景：窗帘、玻璃、雨天、逆光、橱窗
- 焦段：长焦或人像模式谨慎使用
- 快门：`1/60s-1/250s`
- ISO：`100-800`
- 曝光：`+0.3EV` 或允许局部高光溢出
- 后期：降低清晰度、降低纹理、统一偏冷或偏粉

### shoegaze 噪声和吞没

- 场景：演出、人群、车灯、强光源、夜晚玻璃
- 焦段：`1x` 或超广角贴近
- 快门：`1/15s-1/4s`
- ISO：`400-1600`
- 曝光：允许高光溢出，但保留暗部骨架
- 后期：颗粒、低饱和、失焦、拖影可以叠加

### 森山大道式粗粝压力

- 场景：街头近距离、强光阴影、黑白也可以
- 焦段：`1x`
- 快门：`1/125s-1/500s`
- ISO：`800+` 可以接受
- 曝光：压暗，保留硬光和黑
- 后期：高反差、粗颗粒、少一点“漂亮”

### Stephen Shore 式日常观看

- 场景：普通街角、路边、室内、餐桌、停车场
- 焦段：`1x` 或轻长焦
- 快门：`1/125s-1/250s`
- ISO：`50-400`
- 曝光：相对准确
- 后期：颜色克制，空间关系清楚，不靠夸张效果

## 手机和相机的差异

手机上要特别记住：

- 手机会自动 HDR、多帧合成、降噪和锐化
- 同样的 ISO 和快门，不一定和相机质感完全一样
- 手机光圈多数不可控，所以你更依赖焦段、距离、对焦、快门、ISO 和后期
- 手机夜景模式可能会“修得太干净”，高级篇有时反而适合普通模式或专业模式

相机上会多出：

- 可控光圈
- 更真实的景深
- 更大的动态范围
- 更好的高 ISO 质量
- 更精确的手动对焦和慢门控制

但底层判断不变：你还是在决定光、时间、清晰度、颜色和代价。

## 我会怎么点评这一章

我会重点看：

1. 你是否能说清每个参数改变了什么
2. 你的参数选择是否服务拍摄目标
3. 你是否能接受并描述参数代价
4. 风格配方是否真的落到拍摄动作
5. 你是否开始从“碰运气”变成“可复现”

## 这一章最重要的一句话

**参数真正有用的时候，是你知道自己想要什么，并且知道为了它愿意牺牲什么。**
