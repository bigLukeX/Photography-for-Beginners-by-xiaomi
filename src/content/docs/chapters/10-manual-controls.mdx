到了这一章，你终于开始正式碰手机里的手动模式。

但我想先提醒你一句：

**参数不是摄影的起点，而是你在有判断之后，拿来更精确控制画面的工具。**

如果前面几章你已经开始会看主体、构图、光线、焦段，这一章就会顺很多。因为你终于知道“为什么要调”，而不是只是“会调”。

## 这一章学完以后，你应该能做到

- 知道快门、ISO、白平衡、对焦距离分别在控制什么
- 分得清曝光补偿和手动曝光的大致边界
- 明白什么叫“压暗一点更有气氛”
- 知道参数不是独立按钮，而是在交换代价
- 开始理解手机手动模式和相机手动模式的共通逻辑

## 先记住一句话

**参数不是为了把一切调到“正确”，而是为了把画面调到“更符合你的意图”。**

所以这一章最重要的不是参数表，而是：

- 你到底想保住什么
- 你愿意接受什么代价
- 你为什么想让这张图更亮或更暗

## 快门：你在决定“时间感”

先把快门理解成：

**你愿意让一瞬间有多短，或者有多长。**

更快的快门通常意味着：

- 更容易定住动作
- 更不容易糊
- 但进光可能更少

更慢的快门通常意味着：

- 更容易拍进光线
- 更容易留下拖影或抖动
- 更适合刻意营造流动感

在手机上，快门的可控空间不一定像相机那么大，但理解这个逻辑非常重要。

| 快门速度 | 适合什么 | 可能的代价 |
| --- | --- | --- |
| `1/1000s` | 强光、快速运动 | 暗光下不够亮 |
| `1/250s` | 白天街拍、普通抓拍 | 夜晚会需要更高 ISO |
| `1/125s` | 人像、慢走人物 | 动作快会糊 |
| `1/60s` | 静物、站定人物 | 手持开始需要稳 |
| `1/15s` | 轻微运动模糊 | 人物和手抖明显 |
| `1/4s` | 拖影、车流、人影 | 需要稳定支撑 |
| `1s+` | 明显慢门 | 基本需要固定手机 |

## ISO：你在决定“亮一点”要不要拿纯净度来换

ISO 最容易被误解成“调亮度”。

更准确地说，它是在问你：

**当光不够时，你要不要用更高的感光代价来换更亮的画面。**

更高 ISO 常见的代价：

- 噪点更多
- 细节更脏
- 纯净度下降

所以不要一觉得暗就无脑把 ISO 往上拉。

| ISO | 画面倾向 | 使用建议 |
| --- | --- | --- |
| `50/100` | 最干净 | 白天、强光、慢门固定机位 |
| `200/400` | 轻微质感 | 室内、阴天、傍晚 |
| `800` | 明显暗光感 | 夜景、街头、lofi 颗粒可以接受 |
| `1600+` | 噪点和涂抹更明显 | 只在你愿意接受粗糙时使用 |

## 白平衡：你在决定画面的冷暖语言

白平衡不只是“颜色准不准”，它也会影响氛围。

这里最容易把两个方向说反，先分清：

- 光源自身的色温：低 Kelvin 的灯光通常更暖、更偏黄红；高 Kelvin 的环境光通常更冷、更偏蓝。
- 专业模式里的 WB 数值：它是在告诉手机“当前光源大约是什么色温”，手机会做反向补偿。因此在同一个场景里，WB 数值调低，画面通常会更冷、更蓝；数值调高，画面通常会更暖、更黄红。

所以不要把 `3000K` 简单背成“暖色滤镜”。在钨丝灯下把 WB 设为约 `3000K`，目的往往是抵消灯光本身的黄色，让白色回到中性。

偏暖的画面常常会让人觉得：

- 更柔和
- 更靠近傍晚
- 更有人情味

偏冷的画面常常会让人觉得：

- 更清爽
- 更安静
- 更像阴天、夜晚或金属空间

所以白平衡不是只有技术性，也有表达性。

| 专业模式 WB 设定 | 同一场景中的画面倾向 | 可以怎么用 |
| --- | --- | --- |
| `2800K-3500K` | 相对更冷、更蓝 | 抵消暖黄灯，也可主动制造冷感 |
| `4000K-5000K` | 中性到略冷 | 混合光、阴天、克制的日常观察 |
| `5200K-6000K` | 中性到略暖 | 日光附近的起点，先观察白色和肤色 |
| `6500K+` | 相对更暖、更黄红 | 暖色记忆、夕阳感，但要防止肤色过黄 |

最稳的练法不是背表，而是在同一现场固定曝光，分别拍低、中、高 3 档 WB，再看白色物体和肤色往哪边变化。

## 对焦距离和测光：你在告诉手机“先管哪里”

很多新手一提参数，只盯着快门和 ISO，但对焦和测光同样重要。

你可以简单理解：

- 对焦是在说：哪里要清楚
- 测光是在说：哪里决定亮暗参考

如果你让手机对错地方对焦、对错地方测光，画面很可能一开始就跑偏了。

对焦和测光可以分开理解：

| 控制 | 你在告诉手机什么 | 常见用法 |
| --- | --- | --- |
| 对焦 | 哪里要清楚 | 点人脸、手、招牌、前景 |
| 测光 | 哪里决定亮暗 | 点人脸保肤色，点灯牌保高光 |
| 对焦锁定 | 清楚位置先不变 | 等人物走进画面，或者拍静物系列 |
| 故意失焦 | 不让信息完全清楚 | 梦境、记忆、shoegaze、人物不完整 |

## 曝光补偿和手动曝光，不是同一种自由度

你可以先这样理解：

- 曝光补偿：在自动逻辑上，稍微告诉手机“请亮一点”或“请暗一点”
- 手动曝光：你自己更明确地控制参数组合

所以曝光补偿更像：

- 我认可自动的大方向
- 但我想让它偏一点

而手动模式更像：

- 这次我不想完全交给自动
- 我想明确换一种画面倾向

## 参数真正难的地方，不是记住，而是接受代价

比如：

- 你想快门更快，可能就得接受更暗或更高 ISO
- 你想更亮，可能就得接受高光更危险
- 你想夜景更纯净，可能就不能乱拍手抖

所以参数不是四个按钮一起发光，而是一种取舍系统。

## 小米 15 Ultra 上怎么练这一章

最推荐的做法不是一上来拍复杂题材，而是先练这些：

- 同一场景拍一个正常版本、一个压暗版本、一个偏亮版本
- 同一静物在不同白平衡倾向下比较气氛
- 夜里用手动模式控制一组“更稳 / 更亮 / 更纯净”的对比

重点是比较差异，而不是一次拍出神图。

等你能说清楚快门、ISO、白平衡和对焦的作用，再去看 [专业模式进阶工具](../../guides/pro-mode-tools/)；那里会继续讲直方图、峰值对焦、RAW/DNG、稳定与文件输出。不要把这些工具和核心参数同时第一次学习。

## 什么时候该压暗一点

这些场景通常特别值得试着压暗一点：

- 夜景
- 逆光
- 灯牌和建筑亮部很多的城市景
- 你想保住高光和氛围的画面

你要慢慢建立一个新习惯：

不是自动给你多亮你就多亮，而是问自己：

“这张图如果暗一点，会不会更像我看到的样子，或者更有气氛？”

## 参数新手最容易犯的两个错

### 1. 觉得手动模式一定更专业

不一定。

很多时候自动模式已经足够，只需要你做一点曝光补偿和更好的构图判断。

### 2. 一上来就同时乱调很多参数

这样你拍完通常分不清：

- 是哪一个参数让画面变了
- 还是只是自己手抖了、光变了

最稳的练法是：

- 一次只刻意观察一个变量

## 交作业前先自检

- 你是不是知道自己为什么调这个参数
- 画面变化是不是有明确理由
- 你有没有开始接受“参数换代价”这件事
- 夜景是不是还保留了夜的感觉
- 你能不能说清楚哪一张更符合自己的意图

## 我会怎么点评这一章

我会重点看：

1. 你是不是开始有意识地用参数服务画面
2. 曝光选择是不是更成熟了
3. 白平衡有没有带来明确氛围差异
4. 你有没有开始理解取舍，而不是只追求更亮
5. 你的参数说明是不是和结果对应得上
